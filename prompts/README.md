# Prompts（工作剧本）

本目录用于沉淀面向 AI/Agent 的工作流剧本。

使用顺序：

- 先按根入口读取规则：[../AGENTS.md](../AGENTS.md)
- 再按任务类型选择对应 workflow：
  - 新功能开发：[feature.md](feature.md)
  - 缺陷修复：[bugfix.md](bugfix.md)
  - 代码评审：[code-review.md](code-review.md)
- 通用同步更新项与验证最低口径见：[shared-checklist.md](shared-checklist.md)；code review 也可将其作为回归与交付检查基线
- 若任务属于新增页面、信息架构调整或关键交互重做，同时进入 [../design/README.md](../design/README.md) 与 [../design/AGENTS.md](../design/AGENTS.md) 完成原型评审流程
